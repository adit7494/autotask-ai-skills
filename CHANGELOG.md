# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-15

### Added
- Initial release with 8 Autotask PSA skills
- `autotask-psa-api` - Core REST API reference (226+ entities, auth, queries)
- `autotask-contract-management` - Recurring services, block hours, retainers
- `autotask-data-warehouse` - SQL Server reporting database integration
- `autotask-execute-command` - Deep-link URLs for Autotask UI
- `autotask-inventory` - Stock levels, POs, serial numbers, transfers
- `autotask-kpi-reporting` - KPI dashboards and analytics patterns
- `autotask-udf` - User-Defined Fields management
- `autotask-webhooks` - Real-time event-driven integrations
- Installation scripts for bash, PowerShell, and batch
- Adapters for Cursor, Windsurf, Continue, Aider, GitHub Copilot
- Usage examples and documentation
